# Dr.Doc Aplikacja Do danych dla danych  
Narzędzie optymalizacji danych według ich wykorzystania

DATA INPUT:   Zbiór danych z różnych źródeł w róźnych formatach i formach
DESTINATION: Cel w jakim te dane mają zostać wykorzystane. 
np. Projekt CzataGPT,Repozytorium Github, Project Brief, Kontekst dla Agenta AI  
DATA OUTPUT: dane podzielone na pliki katalogi odpowiedniu ustrukturyzowane
